# Contributing

Thanks for helping improve `react-chunked-upload`.

## Before you start

- Search existing issues before opening a new one.
- Use the bug report template for reproducible defects.
- Use the feature request template for proposed API changes.
- Keep pull requests focused on one behavior.

## Local development

```bash
npm install
npm test
npm run build
```

To run the example app:

```bash
cd example
npm install
npm run dev
```

## Pull requests

Include tests for behavior changes and update the README when the public API changes. Do not introduce breaking changes without discussing them in an issue first.
